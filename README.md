# Project Overview
ThoughtForFoods is a recipe app mainly focused on searching recipes you can make with ingredients you already have. The application was developed natively for Android using Android Studio as the primary development environment. The frontend interface was built using XML for layout design, while the backend logic was implemented in Java, handling basic mocking functionalities.

## Core Features
* **Pantry Management**: User can add/delete their ingredients in the Pantry, this allow them to use this app as a pantyr management, keeping track of what they have and what they don't.
* **Recipe Search based on Ingredients**: Recipe page show user recipes they can make, ranking from number of ingredient user is missing, from least to most. There's a toggle to only show recipe that user have all the ingredients
